📊 Superstore Sales Analysis (Power BI + Python)

📌 Project Overview

This project combines **Power BI dashboarding** with **Python-based data analysis** to explore the Superstore dataset.

It demonstrates both:

* Business Intelligence (Power BI)
* Data Analysis using Python

---

🛠 Tools & Technologies

* Power BI
* Python
* Pandas
* Matplotlib

---

📂 Project Structure

* `dashboard/` → Power BI dashboard (.pbix)
* `python/` → Python analysis script
* `data/` → Dataset
* `screenshots/` → Dashboard preview

---

📊 Power BI Dashboard Features

* KPI Cards (Sales, Profit, Quantity)
* Sales by Category
* Profit by Region
* Discount vs Profit
* Top Sub-Categories
* Interactive Filters

---

🐍 Python Analysis

Python script replicates dashboard insights:

* Data cleaning using Pandas
* Aggregations and grouping
* Visualization using Matplotlib
